Our Product

The Eye Robot is a visual impairment assistant that utilizes object detection, distance estimation, and text-to-speech (TTS) to help its users navigate through indoor settings. Two versions of the applications are available for download: Eye Robot Pro and Eye Robot Lite. The Eye Robot Pro supports all 3 capabilities and is available for download on iPhone Pro Max models. The Eye Robot Lite only supports 2 capabilities, object detection and TTS, and is available for download on all other iPhone models. This is due to the fact that only the iPhone Pro model supports Lidar, which provides the distance functionality element.
